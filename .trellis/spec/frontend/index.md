# Frontend Development Guidelines

> Current project reality: the frontend is a static site under `site/`.
> Interactivity is vanilla JavaScript embedded in generated HTML. Styling is
> embedded CSS. There is no React, Vue, TypeScript, bundler, package manager, or
> component framework.

---

## Scope

Use these specs when a task changes:

- `site/index.html`
- `site/projects/<slug>/index.html`
- `tools/generate_project_index.py`
- report-page HTML/CSS/JS emitted by `tools/generate_final_insight_reports.py`
- `tools/verify_reports.js`

## Evidence Baseline

- `site/index.html` is generated from `tools/generate_project_index.py`.
- Project report pages under `site/projects/<slug>/index.html` are generated by
  Python report generators or copied by the registration CLI.
- `tools/verify_reports.js` checks rendered pages across desktop, poster, and
  mobile viewports.
